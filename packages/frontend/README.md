# Frontend

Next.js + TypeScript フロントエンドアプリケーション

## 技術スタック

- Next.js 14
- TypeScript
- React 18
- Jest（テストフレームワーク）
- React Testing Library

## セットアップ

```bash
npm install
```

## 開発

```bash
npm run dev
```

## テスト

```bash
# テスト実行
npm test

# カバレッジ付きテスト
npm run test:coverage

# ウォッチモード
npm run test:watch
```
